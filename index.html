CSS Position

The CSS position property defines the position of an element in a document. There are five values the position property.They are-static, relative,absolute,fixed,sticky etc.

Promise In JS

Promises are important building blocks for asynchronous operations in JavaScript and A Promise is a JavaScript object that links producing code and consuming code. such as Pending, Fulfilled,Rejected. While a Promise object is "pending" (working), the result is undefined. When a Promise object is "fulfilled", the result is a value. When a Promise object is "rejected", the result is an error object.
